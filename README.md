SETTING UP:
Click "Select Server Folder" and then select your minecraft server folder:
setupWizard
If you have multiple jars, select the jar you want to use (e.g. "paper.jar") and select the RAM allocation (e.g. 4gb). And then click the white button:
setupOptions
| You will now be greeted with the dashboard:
dashboard

| From here you can now basically do anything, it's quite simple. Start will launch your server and all buttons will do as expected. There is a live console which you can send commands with and receive commands.

Hope you enjoy, there may be some glitched as we are in an early stage but all shall be fixed soon. Hopefully this helps you!
