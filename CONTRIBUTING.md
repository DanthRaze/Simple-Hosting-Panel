## Code of Conduct
By participating in this project, you agree to abide by our [CODE_OF_CONDUCT.md](https://github.com/DanthRaze/Simple-Hosting-Panel/commit/a68c2d619e22fc5eea05435154f88520ce3ace69). Please be respectful and professional in all interactions.

## How Can I Contribute?

### Reporting Bugs
* **Search first:** Check if the bug has already been reported in the Issues tab.
* **Be specific:** Include steps to reproduce, the expected behavior, and what actually happened.
* **Provide context:** Mention your OS, version, and any relevant logs.

### Suggesting Enhancements
* Open a new issue and describe the feature you'd like to see.
* Explain why this feature would be useful to most users.

### Pull Requests
1. **Fork** the repository and create your branch from `main`.
2. **Install dependencies** and ensure the project runs locally.
3. **Make your changes.** If you've added code, please add tests!
4. **Follow coding styles:** Ensure your code matches the existing style (indentation, naming, etc.).
5. **Write a good commit message:** Use the imperative mood (e.g., "Add logging to auth module").
6. **Submit the PR.** Reference any related issues in the description.
